# fastapi-proj-boilerplate

## Prerequisites
- python 3.11 up
- [docker](https://docs.docker.com/desktop/install/mac-install/)
- (optional) [httpie](https://httpie.io/docs/cli/installation)


## Examples

### Build service

```shell
make build
```

### Run service

```shell
make run
```

### Test service

```shell
make test
```
